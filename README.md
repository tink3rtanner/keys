# keys

Public JWKS for Epic on FHIR backend OAuth 2.0 testing.

- `jwks.json` — RS384 RSA-2048 public key
- `.well-known/jwks.json` — same, conventional path
- Algorithm: `RS384`
- kid: see the `kid` field inside `jwks.json`

Served via GitHub Pages at https://tink3rtanner.github.io/keys/jwks.json
